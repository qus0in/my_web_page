#### 개요
CampAI 프로젝트는 딥러닝을 이용해 캠핑장 이미지를 분류하는 프로젝트입니다. 이 프로젝트에서는 ResNet, EfficientNet, VGG 등의 딥러닝 아키텍처를 사용하여, 이미지 분류 모델을 구성합니다. 또한, Streamlit과 Colab을 사용하여 데이터 시각화 및 분석을 수행합니다.
